Git

git init Add GitHub repository git push commit status add reset log checkout diff Delete from working and staging area Create a new branch merge Delete branch stash Download from GitHub Delete previous commits Remove deleted commits from Github

• Initialize a repository => git init

• Add a new remote repository with github => git remote add origin https://github.com/yourusername/your-repo.git

• First time pushing to github => git push -u origin master (after just use => git push)

• Taking a snapshot => git commit -m “message” (git commit -a -m => commits and stages everything)

• How things are => git status

• Add files to staging area where they are tracked by git but not committed yet (no snapshot yet) => git add file (git add . => add all new and modified files)

• To remove all the files from staging area after git add => git reset

• See what commits have been submitted by who and when including the commit message => git log

• View all the currently existing branches => git log -- file

• See what commits have been submitted for a specific file => git log -- file

• See a commit graph. All commits line by line and branching. Also shows where the head is pointing at => git log --all --decorate --oneline --graph

• Get a deleted file back to a working directory and tracked directory from a previous commit => git checkout 9ddb9 -- file

• View the difference between tracked files and the working directory. Will not show untracked files or committed files. Red means removed, green means added => git diff (git diff --staged => difference between tracked files aka staging area vs most recent commit aka snapshot)

• See what will be changed when a branch is merged into master => git diff master..branch-name

• To see changes between any file and any previous commit => git diff commit-hash file-name (can take other arguments)

• Delete a file from working area and staging area aka tracked by git => git rm file

• Create a new branch or switch to another branch => git branch branch-name

• To create and checkout a new branch at the same time => git checkout -b branch-name

• See what commits have been submitted for a specific file => git branch

• To merge a branch with master => git merge branch-name

• To see what branches have been merged with master => git branch --merged

• To delete a branch when it is merged => git branch -d branch-name

• To delete a branch that is not merged yet => git branch -D branch-name

• Save changes to a working tree temporarily and locally in a separate area => git stash

• See the stashes => git stash list (git stash list -p => see the edits made in each stash)

• Reapply the most recent stash => git stash apply (git stash pop => apply the stash and remove it)

• To apply a specific stash => git stash apply label

• To download from GitHub => git clone path

• To delete previous commits from git => git reset --hard HEAD~1 or git reset --hard sha1-commit-id

• To remove the deleted commits in github => git push origin HEAD --force
