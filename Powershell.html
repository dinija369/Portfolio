Powershell

Functions and parameters Test-Path Out-File Set-Content Get-ChildItem [files].Count ForEachObject Where-Object Sort-Object Join-Path

• Create a function => function [name] {…} Declare parameters to be passed in the function call and used in function => param ( [data-type]$name [string]$path )

• Test path cmdlet checks if the path exists and if it leads to something => Test-Path -Path $path ( Test-Path -Path $path -PathType Container => will tell if the path contains directories == Container, files == Leaf or both == Any)

• To create a new file in the current directory and write something to it => Out-File -FilePath [name.txt] ( Out-File -FilePath [name.txt] -NoClobber => can be added to not overwrite already existing files, Out-File -Append -FilePath [path] => -Append can be added to not overwrite existing content but to add content at the end)

• Create a new file in current directory that has a value => Set-Content -Path [name.txt] -Value [string] (this will create a new document entirely or overwrite previous content)

• Get items in a specific location => Get-ChildItem -Path [path] ( add -File == for only files, -Directory == for only folders).

• Count the files and check if the count is 0 => [files].Count -eq 0

• To iterate through each file => [collection of input objects aka files here] | ForEachObject {….} ( {…} => is a script block that can be used to specify the operation). ( { $_.Name | Out-File -Append -FilePath $reportFilePath } => name specifies the file name. $_ => specifies the current object and the action performed on the object after the pipe.)

• Filter objects => $files | Where-Object { $_.LastWriteTime -ge $parsedDateTime } (-ge == greater then or equel)

• Sort objects => $filteredFiles | Sort-Object -Property Name -Descending

• Combine the current folder's path with the filename 'Report.txt' to create a complete file path => Join-Path -Path (Get-Location) -ChildPath "Report.txt"
